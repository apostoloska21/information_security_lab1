Лабораториска 1: Корисничка Автентикација со Flask и PostgreSQL
Преглед
Веб-апликација за безбедна корисничка автентикација со имплементација на основни концепти од информациската безбедност. Апликацијата овозможува регистрација, пријава (логирање), и одјава (логаут) на корисници, со фокус на безбедносна заштита од XSS, SQL Injection и Session Hijacking напади.

Цел и Задачи
✅ Регистрација на нови корисници – Валидација на email и лозинка
✅ Пријава и сесиски менаџмент – Flask-Login за управување со сесии
✅ Хеширање на лозинки со salt – SHA-256 со уникатен salt за секој корисник
✅ XSS заштита – Санитизирање на кориснички input
✅ SQL Injection заштита – SQLAlchemy со parameterized queries
✅ Session Cookie заштита – HttpOnly, Secure
✅ PostgreSQL база – Складирање на кориснички податоци

Користени Технологии
Backend: Python 3.x, Flask

Датабаза: PostgreSQL 15.x

ORM: SQLAlchemy

Управување со сесии: Flask-Login

Хеширање: hashlib (SHA-256) со salt

Заштита: XSS санитизирање, CSRF заштита

Frontend: HTML5, Jinja2 templates
